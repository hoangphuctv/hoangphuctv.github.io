Ubuntu Cài đặt VMWare 7.1 cho Linux

Oct 10, 2011 · 2 minute read
Ubuntu

       Đối với các bạn IT thì việc sử dụng các phần mềm tạo máy ảo là không thể thiếu được. Mặc dù VirtualBox đã quá tốt cho việc tạo và sử dụng máy ảo trên nền nảng Linux rồi, nhưng hôm nay mình sẽ giới thiệu đến các bạn cách cài đặt VMWare - phần mềm tạo máy ảo mà đa số chúng ta sử dụng trên HĐH Windows.
       Không có gì phải lúng túng khi làm quen với một nền tảng mới cả, cách cài đặt VMWare cũng đơn giản như trên Windows vậy.
       Giờ thì bạn hãy vào trang downloads của VMWare để tải bản VMWare dành cho bản Linux của mình (32bit hay 64bit) Tất cả các distro Linux đều sử dụng chung các file cài đặt này. File tải về có dạng *.bunble, dung lượng ~309MB :D
       Trong khi chờ đợi tải file đó về bạn hãy vào terminal gõ lệ [code]sudo apt-get install build-essential linux-headers-$(uname -r)[/code]
       Sau khi tải xong file 309MB ở trên, thì bạn thêm quyền thực thi cho nó. Ví dụ bạn tải nó về đặt trong ~/Downloads và nó có tên VMware-Workstation-Full-7.1.3-324285.i386.bundle thì gõ lệnh [code]chmod +x ~/Downloads/VMware-Workstation-Full-7.1.3-324285.i386.bundle[/code]
       Tiếp tục lệnh: [code]sudo ./Downloads/VMware-Workstation-Full-7.1.3-324285.i386.bundle[/code]
       Thế là màn hình sau đây xuất hiện, bạn chỉ cần tiến hành cài đặt như trên Windows qua các bước mà Wizard hướng dẫn.






Ở bước sau đây bạn không cần phải điền gì cả (nếu bạn muốn thay đổi vị trí cài đặt thì hãy gõ đường dẫn vào - tuy nhiên not recommend)
