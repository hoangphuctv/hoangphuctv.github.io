Chuyển đổi qua lại các gói giao diện của Ubuntu

Dec 18, 2011 · 1 minute read
Ubuntu

Bạn đang sử dụng Ubuntu, thấy hơi chán với cái Utiny nặng nề, muốn chuyển sang xubuntu (sử dụng xfce) cho nhẹ, hoặc chuyển sang kubuntu cho màn hình bóng bẩy hơn! Hoặc chuyển ngược lại. Không cần phải cài đặt lại các bản Ubuntu, Xubuntu hay Kubuntu tương ứng, mà chỉ cần cài lại gói giao diện cho nó là OK! Việc thực hiện chỉ đơn giản như sau:

  • Nếu muốn chuyển sang Xubuntu:sudo apt-get install xubuntu-desktop
  • Nếu muốn chuyển sang Edubuntu:sudo apt-get install edubuntu-desktop
  • Nếu muốn chuyển sang Kubuntu:sudo apt-get install kubuntu-desktop
  • Nếu muốn chuyển sang Ubuntu:sudo apt-get install ubuntu-desktop
    Chỉ đơn giản như vậy! Và khi muốn gỡ nó ra thì cũng chỉ gõ các lệnh tương ứng:

Chúc các bạn thành công và iu thix Ubuntu hơn nữa :x !
