Phần mềm cắt nối file 001 | 002 .. trên Linux

Jul 2, 2011 · 1 minute read
Công cụLinux

Các file 001,… đối với các bạn đã quá quen thuộc, thế nhưng trên nền Linux bạn đang loay hoay tìm một ứng dụng thay thế?? lxspit là một ứng cử viên sáng giá. Mặc dù chương trình không có giao diện (sử dụng command) thế nhưng lxspit cũng tỏ ra vô cùng mạnh mẽ!! Các lệnh với xlspit cũng rất đơn giản. Cấu trúc lệnh:
[code]$ lxspit <-option>

Để hiểu rõ hơn bạn có thể thao khảo thêm lệnh:
[code]$ man lxspit[/code]
