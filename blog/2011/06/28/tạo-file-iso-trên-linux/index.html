Tạo file Iso trên Linux

Jun 28, 2011 · 1 minute read

Chào các bạn! Hôm nay mình sẽ tiếp tục chinh phục Linux bằng việc tìm hiểu cách tạo file ISO.
Như bạn đã biết trên Windows muốn tạo file iso thì chúng ta chỉ việc tìm phần nào đó hỗ trợ tạo file iso (đại loại như Ultra ISO) là có thể tạo được, nó khá đơn giản. Thế còn trên Linux thì sao?? Thật sự mà nói. Nó còn đơn giản hơn nhiều ^^!.
Việc đầu tiên là bạn phải xác định xem Linux nhận ổ đĩa CD của bạn là thiết bị gì?. Thông thường là cdrom0, sr0, scd0.
Các bạn lần lượt gõ lệnh như hình để kiểm tra.


Như hình trên thì ổ cd máy mình sr0 và scd0. Bây giờ, tiếp tục việc tạo file iso chỉ bằng một lệnh !
[code] sudo cp /dev/sr0   ~/tenfile.iso[/code]
Chờ trong giây lát là bạn đã có file iso như ý [;)]
