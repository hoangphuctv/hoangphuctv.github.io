Code C/C++ Tam giác Pascal

Jun 23, 2011 · 1 minute read

Đây là bài code Tam giác Pascal kinh điển. Mình lấy nguồn từ forum của lớp 08CT Hùng Vương.
(Có chỉnh sửa đôi chút để chạy được trên Linux ^^!)

#include <stdio.h>

#define N 15 //So luong dong trong tam giac pascal
void main() {
  int i, j, tgpascal[N], tmp[N];
  tgpascal[0] = 1;
  for (i = 1; i < N; i++) tgpascal[i] = 0;
  for (i = 0; i < N; i++) {
    for (j = 0; j <= i; j++) tmp[j] = tgpascal[j];
    printf("n%5d", tmp[0]);
    for (j = 1; j <= i; j++) {
      tmp[j] += tgpascal[j - 1];
      printf("%5d", tmp[j]);
    }
    for (j = 0; j <= i; j++) tgpascal[j] = tmp[j];
  }
  printf("n");
}
