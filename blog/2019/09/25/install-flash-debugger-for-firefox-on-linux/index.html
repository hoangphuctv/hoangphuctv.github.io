Install Flash Debugger for Firefox on Linux

Sep 25, 2019 · 1 minute read

Thời điểm hiện tại chắc có rất ít bạn còn “cho phép flash chạy” trên trình duyệt của mình. Còn ít hơn nữa những bạn làm việc trên các project flash. Flash hiện tại đang bị giới hạn ở mọi nơi. Và ngày lên đoạn đầu đài đã được định:

  • Adobe: https://theblog.adobe.com/adobe-flash-update/
  • Apple: https://webkit.org/blog/7839/adobe-announces-flash-distribution-and-updates-to-end/
  • Facebook: https://developers.facebook.com/blog/post/2017/07/25/Games-Migration-to-Open-Web-Standards/
  • Google: https://www.blog.google/products/chrome/saying-goodbye-flash-chrome/
  • Microsoft: https://blogs.windows.com/msedgedev/2017/07/25/flash-on-windows-timeline/
  • Và cả Mozilla: https://blog.mozilla.org/futurereleases/2017/07/25/firefox-roadmap-flash-end-life/

Nhưng,… Nếu,… đến giờ mà bạn vẫn còn maintain project flash thì sao :D, mà lại còn là tay ngang nữa.

via GIPHY

Hy vọng bài viết có thể giúp phần nào cho bạn.

1./ Tải về flash debugger dành cho firefox

https://fpdownload.macromedia.com/pub/flashplayer/updaters/32/flash_player_npapi_linux_debug.x86_64.tar.gz

  • Giải nén ra thư mục ~/Download/flash_player_npapi_linux_debug.x86_64 (bước này cứ nhắp phải file vừa tải về. rồi chọn Extract here...)

  • Move to /usr/lib:

    $ sudo mv  ~/Download/flash_player_npapi_linux_debug.x86_64 /usr/lib/

2./ Remove old version or non-debugger flash

$ sudo rm /usr/lib/mozilla/plugins/flashplugin-alternative.so

3./ Link to plugin

$ sudo ln -s /usr/lib/flash_player_npapi_linux_debug.x86_64/libflashplayer.so /usr/lib/mozilla/plugins/flashplugin-alternative.so

4./ Restart firefox

Thoát hoàn toàn firefox. Và mở lại.

Nhắp phải tại flash nếu thấy thêm 2 item menu nữa là bạn đã cài đặt thành công.
