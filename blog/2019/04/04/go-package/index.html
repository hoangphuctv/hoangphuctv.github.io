Lập trình GO

Chương trình go được tạo bởi các package. Trong đó, hàm main trong package có tên là main sẽ được tự động thực thi. Tức là một package muốn thực thi được, thì phải là package main.

Nếu chúng ta cố gắng chạy 1 package không phải là package main thì go sẽ báo như sau:

Tạo file test.go:

package test

func start(){
    print("Testing...")
}





$ go run test.go
go run: cannot run non-main package
