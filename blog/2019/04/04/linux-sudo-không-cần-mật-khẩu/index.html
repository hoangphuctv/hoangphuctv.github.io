Linux sudo không cần mật khẩu

Apr 4, 2019 · 1 minute read
Linuxsudo

Trên HĐH Linux, bạn mệt mỏi phải nhập mật khẩu mỗi khi sudo mỗi khi cần gõ lệnh nào đó cần quyền root. Bạn có biết là bạn có thể sudo mà không cần nhập mật khẩu.

Cách thực hiện: Ví dụ tên username là: phucth 

sudo vim /etc/sudoers.d/phucth
Tên vào file nội dung như sau: 
phucth ALL=(ALL:ALL) NOPASSWD: ALL
Lưu lại. Tiến hành kiểm tra: 
$ sudo ls /
Bây giờ, lệnh ls thực hiện được dưới lệnh sudo mà không cần nhập nữa.

LƯU Ý:

  1. User của bạn phải là sudoer trước đó (user có thể sudo được)

  2. Khi bạn bật tính năng này user của bạn có quyền sinh sát “như root” gõ lệnh enter là ăn luôn. Không có thời gian “nghĩ lại” trong lúc bạn nhập mật khẩu. Vì vậy nên cẩn thận.
