Cài đặt brew trên HĐH MacOS

Apr 7, 2019 · 2 minute read
brewCông cụinstallMacOS

brew là một trình quản lý gói dành cho HĐH MacOS nó giống như yum, hay apt trên linux.

Yêu cầu hệ thống: Máy Mac đang chạy CPU Intel HĐH OS X 10.11 hoặc cao hơn. Chạy lệnh cài đặt trên các shell bash hoặc zsh. Các shell notably, fish, tcsh và csh sẽ bị lỗi khi cài đặt. Nếu máy bạn chưa cài đặt Command Line Tools (CLT) thì cần cài đặt trước

$ xcode-select --install

Bắt đầu cài đặt brew:

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Các lệnh trên chỉ cần chạy ở user mode. Không cần sudo hoặc quyền root. Sau khi chạy lệnh trên, bạn thử xem có cài đặt thành công chưa bằng lệnh:

$ brew -v
Homebrew 1.6.9
Homebrew/homebrew-core (git revision b577e; last commit 2018-07-07)

Nếu brew trả về kết quả tương tự như trên là bạn đã cài đặt thành công.

Cài đặt phần mềm

Ví dụ như cài wget:

$ brew install wget

Lệnh trên sẽ cài đặt gói wget. Bạn thay thế wget thành tên của gói bạn muốn cài đặt.

Tìm kiếm phần mềm

Có những gói phần mềm có tên hoặc nhiều phiên bản. Ví dụ như có thể bạn đang cần cài đặt php7.0 nhưng không biết nên gõ thế nào cho đúng. Bạn gõ brew install php70 mà brew lại báo lỗi Error: No available formula with the name "xxxx". Vì tên của gói php7.0 nó là php@7.1. Vậy nên, bạn nên tìm để xem chính xác tên đó như thế nào. Sau đó chúng ta cài đặt ứng với tên tìm được. Chúng ta tìm bằng từ khoá search như sau:

$  brew search php 
==> Formulae
brew-php-switcher   php-cs-fixer        php@7.1             phpunit
php                 php@5.6            phplint
php-code-sniffer    php@7.0            phpmyadmin

==> Casks
eclipse-php                netbeans-php               phpstorm

Vậy bây giờ bạn đã thấy cách ghi chính xác tên của php 7.0 là php@7.0. Bạn cài đặt tương tự như lệnh cài đặt trên.

$ brew install php@7.0

Nếu gặp khó khăn gì bạn vui lòng comment bên dưới. Chúc bạn cài đặt thành công!
