Linux tìm file thư mục với lệnh find

Apr 7, 2019 · 2 minute read
Công cụfindLinuxtouch

find là một công cụ mạnh mẽ giúp chúng ta tìm kiếm các tập tin và thư mục trên hệ thống một cách nhanh chóng.

Tìm không có điều kiện

Tìm không có điều kiện nào, tức là sẽ list tất cả các file và thư mục:

$ find .

Dấu . tức là thư mục hiện tại, nếu bạn muốn tìm trong 1 thư mục khác, không phải thư mục đang đứng thì bạn gõ vào đường dẫn đến thư mục đó.

$ find /home/phucth/Downloads/

Tìm tất cả file hoặc thư mục:

  • Nếu tìm tất cả các file gõ: find . -type f
  • Nếu tìm tất cả cá thư mục gõ: find . -type d

Tìm theo định dạng tập tin:

Tìm tất cả các file .log ta gõ:

$ find . -name "*.log"

Tìm trong thời gian 1 ngày:

Dựa vào lệnh find cho phép chúng ta tìm kiếm các file theo thời gian của 1 file mẫu. Nên chúng ta sẽ thực hiện các bước sau:

  • Tạo 1 cái file bằng lệnh touch với thời gian mà mình mong muốn tìm. Ví dụ 1 ngày. File này chỉ là file tạm.
  • Dùng lệnh find để tìm với cờ -newer.
  • Xóa cái file tạm.

Lệnh đầy đủ như sau:

touch -d '1 day ago' .cutoff && find . -type f -newer .cutoff  && rm .cutoff
