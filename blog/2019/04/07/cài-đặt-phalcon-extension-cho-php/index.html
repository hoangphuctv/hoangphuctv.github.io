Cài đặt phalcon extension cho php

Apr 7, 2019 · 2 minute read

Hướng dẫn cài đặt phalcon extension cho php

Môi trường:

  • Ubuntu 18.04
  • PHP 7.2

Cài đặt

Cài thư viện:

apt install gcc libpcre3-dev php-dev

Clone source: ta lưu source tại /usr/local/src/

cd /usr/local/src/
git clone --depth=1 "git://github.com/phalcon/cphalcon.git"
sudo ./install

Chờ một chút lệnh sẽ chạy thành công. Để lưu đúng cấu trúc của ext php ta tạo mới file cấu hình ini tại: /etc/php/7.2/mods-available/phalcon.ini

Nội dung file:

extension=phalcon.so

Để bật ext phalcon cho fpm ta tạo link như sau:

# ln -s /etc/php/7.2/mods-available/phalcon.ini /etc/php/7.2/fpm/conf.d/90-phalcon.ini

Để có tác dụng ta cần restart fpm:

# /etc/init.d/php7.2-fpm restart

Để bật ext trên cli ta tạo link:

ln -s /etc/php/7.2/mods-available/phalcon.ini /etc/php/7.2/cli/conf.d/90-phalcon.ini

Sau khi tại link xong, để biết có bật ext thành công không ta gõ lệnh:

Gỡ lỗi

Nếu báo lỗi như sau:

PHP Fatal error: Class ‘jsonserializable’ not found in Unknown on line 0

Thì phalcon đang được load trước các thư viện cần thiết. Chúng ta tăng con số của file link lên. Ở trên mình đã đặt 1 con số khá to là 90 rồi. Nên chắc không lỗi. Nhưng nếu vẫn bị lỗi, bạn tăng con số này lên nữa. Ví dụ: 99-phalcon.ini.

Phải xóa file cũ trước: rm /etc/php/7.2/fpm/conf.d/99-phalcon.ini

ln -s /etc/php/7.2/mods-available/phalcon.ini /etc/php/7.2/fpm/conf.d/99-phalcon.ini
