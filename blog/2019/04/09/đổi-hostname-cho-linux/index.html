Đổi hostname cho Linux

Apr 9, 2019 · 1 minute read
Cài đặt &amp; Cấu hìnhhostnameLinuxLinux

Hostname là tên, dùng để đặt cho máy tính. Giúp cho người quản trị ghi nhớ dễ dàng, và phân biệt các máy mới với nhau. Để xem hostname hiện tại bạn nhìn trên terminal các bạn sẽ thấy hostname tại vị trí dấu nhắc lệnh:

root@myserver #

Nếu không thấy hostname tại vị trí dấu nhắc lệnh, bạn có thể gõ lệnh sau để xem hostname:

$ hostname
myserver

Muốn thay đổi myserver này các bạn chỉ cần chỉnh sửa file  /etc/hostname Để xem nội dung hiện tại bạn gõ lệnh:

$ cat /etc/hostname
myserver

Bạn thấy nội dung hiện tại là “myserver” Bạn chỉ cần đổi nội dung file trên thành tên hostname bạn mong muốn

$ sudo vi /etc/hostname

bấm phím i để vào chế độ edit của vim đổi nội dung file thành tên hostname mong muốn: backup-server

Bấm :x để lưu file. Và reboot máy bằng  lệnh:

$ sudo init 6

Sau khi máy khởi động lại xong, bạn sẽ thấy hostname thay đổi. Check lại tên hostname bằng lệnh:

$ hostname
backup-server
