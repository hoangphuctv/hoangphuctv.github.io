Cài extension PHP còn thiếu trên Linux

Apr 1, 2019 · 1 minute read
apache2Cài đặt - Cấu hìnhextensioninstall

Khi làm việc với php bạn thường hay gặp các lỗi Undefined function, hoặc class thường là do thiếu các extension gây ra. Ví dụ như “undefined function curl_init()” là do thiếu extension curl.

Cài thông qua trình quản lí gói

Nếu bạn cài đặt php thông qua trình quản lí gói như apt, bạn chỉ cần cài thêm extension qua lệnh:

$ apt install php-curl

Cài thông qua tool pecl

Còn nếu như bạn cài thông qua build source, thì bạn có thể cd vào thư mục bin của php ví dụ: /etc/php/5.6/bin. Tiến hành cài bằng tool pecl:

$ cd /etc/php/5.6/bin
$ ./pecl install curl

Xem thêm bài viết: Build extension cho php bất kì trên Linux/MacOS

Sau khi cài đặt

Sau khi cài đặt thành công. Sau khi cài đặt thành công bạn cần:

Nếu chạy mod_php qua apache2, thì cần restart apache2 bằng lệnh:

$ sudo systemctl restart apache2

Nếu bạn chạy php-fpm thì restart bằng lệnh:

$ sudo /etc/init.d/php-fpm restart

Nếu không có script này thì bạn tìm script có tên gần giống, vì mỗi phiên bản có thể sẽ đặt tên khác nhau:
