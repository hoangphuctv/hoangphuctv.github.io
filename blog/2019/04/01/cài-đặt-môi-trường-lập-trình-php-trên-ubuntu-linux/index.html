Cài đặt môi trường lập trình PHP trên Ubuntu Linux

Apr 1, 2019 · 2 minute read
apache2Cài đặt - Cấu hìnhinstallLinuxmysqlphp

Để lập trình PHP trên HĐH Ubuntu, chúng ta cần cài đặt cơ bản các gói sau:

  1. Web server: Apache2
  2. DB: MySQL
  3. PHP và các extension
  4. Cài đặt text editor

Cài đặt Apache2

Trước tiên, update repo:

$ sudo apt update

Tiến hành cài apache2

$ sudo apt install apache2 -y

Mở trình duyệt vào trang chủ http://localhost. Nếu thấy trang** apache2 default page **là cài đặt thành công. Nếu không vào được localhost thì thử restart lại apache2 bằng lệnh

$ sudo systemctl restart apache2

Document root mặc định của apache2 nằm tại: /var/www/html

Cài đặt MySQL

Cài đặt MySQL server:

$ sudo apt install mysql-server

Sau khi cài đặt xong, muốn tuỳ chỉnh mật khẩu vào mysql chỉ cần sudo lên root sau đó tạo user như mong muốn. Ví dụ tạo user root, mật khẩu 123456.

$ sudo mysql

Khi vào được dấu nhắc lệnh của mysql> bạn gõ tiếp lệnh tạo user sau đây. Đổi mật khẩu thành mật khẩu của bạn mong muốn.

CREATE USER 'root'@'127.0.0.1';
GRANT ALL PRIVILEGES ON *.* To 'root'@'127.0.0.1' IDENTIFIED BY '123456';
FLUSH PRIVILEGES;

Ctrl+D để thoát ra khỏi mysql. Để kiểm tra user vừa tạo đã vào được chưa ta tiến hành test thử:

$ mysql -h 127.0.0.1 -u root -p123456

Khi vào được dấu nhắc lệnh của mysql> là bạn đã tạo tài khoản mysql thành công.

Cài đặt PHP

Để cài đặt bản php mới nhất và stable bạn gõ lệnh:

$ apt install php libapache2-mod-php php-mcrypt php-mysql -y

Sau khi cài đặt php, bạn gõ thử lệnh sau. Nếu ra kết quả là đã cài đặt thành công.

$ php -v
PHP 7.1.23 (cli) (built: Feb 22 2019 22:08:13) ( NTS )
Copyright (c) 1997-2018 The PHP Group
Zend Engine v3.1.0, Copyright (c) 1998-2018 Zend Technologies

Sau đó, khởi động lại apache2

$ sudo systemctl restart apache2

Bạn tạo thử file php info tại: /var/www/html/info.php có nội dung như sau:

Và vào đường dẫn http://localhost/info.php

Thông tin php info sẽ hiện ra tại đây.

Cần cài thêm các extension khác xem ở đây: Cài extension PHP còn thiếu trên Linux

Cài đặt text editor

Có một số editor trên môi trường Linux bạn có thể sử dụng:

  • Vim: có sẵn không cần cài đặt
  • Gedit: có sẵn không cần cài đặt
  • Sublime text: download tại https://www.sublimetext.com/3
  • Visual Code: https://code.visualstudio.com/

Bạn tải về file “xxx.deb” cài đặt bằng lệnh:

$ sudo dpkg -i ~/Download/xxx.deb
