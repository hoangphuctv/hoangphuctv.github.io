Xử lí JSON trong PHP

Apr 18, 2019 · 1 minute read
jsonLập trình PHPphp

JSON là một kiểu định dạng data interchange (trao đổi dữ liệu) phổ biến hiện nay. Các ngôn ngữ lập trình đều hỗ trợ xử lí JSON.
Với PHP việc xử lí rất đơn giản và dễ dàng. Từ array hoặc object ta có thể chuyển đổi dễ dàng qua JSON bằng cặp hàm hỗ trợ encode và decode json.

Ta xét mảng sau đây:

$user = [
    "id"       => 1,
    "username" => "admin",
    "role_id"  => 999
];

Để chuyển mảng trên thành chuỗi JSON ta thực hiện hàm json_encode:

$user_json = json_encode($user);
echo $user_json;





{"id":1,"username":"admin","role_id":999}

Và ngược lại, ta cần decode json ta thực hiện bằng hàm: json_decode:

$user_json = '{"id":1,"username":"admin","role_id":999}'
$user = json_decode($user_json);
var_dump($user);

Hàm json_decode với tham số thứ 2 nếu là TRUE thì dữ liệu trả về sau decode là array, nếu là FALSE thì kiểu dữ liệu trả về là object. Mặc định là FALSE.

**Lưu ý: **nếu chuỗi đầu vào không phải là dạng json string hoặc có độ sâu quá lớn thì hàm json_decode không thể decode được. Hàm sẽ trả về kết quả là NULL.

$str = "Normal text";
$result = json_decode($str);
var_dump($result);
// NULL
