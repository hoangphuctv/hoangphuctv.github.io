Ứng dụng nhanh docker trong công việc hằng ngày

Apr 8, 2019 · 1 minute read
Cài đặt - Cấu hìnhdockermemcacheredis

Tình huống là: Khi cần sử dụng nhanh 1 service nào đó mà không muốn tốn thời gian cài đặt cấu hình. Xài docker chỉ vài lệnh là có thể sử dụng được service mình cần ngay lập tức: Ví dụ memcache. Không cần tải service memcache về cài đặt chỉ cần gõ lệnh:

docker run --name my-memcache -p 11211:11211 -d memcached

Những lần sau cần xài memcache gõ lệnh:

docker start  my-memcache

Khi không cần sử dụng memcached nữa thì xoá đi bằng lệnh:

docker rm  my-memcache
  • Tương tự cho redis port 6380:
# chạy lần đầu tiên
docker run --name my-redis -p 6380:6379 -d redis

# Lần sau cần sử dụng gõ:
dockker start my-redis

# Không cần dùng nữa
dockker stop my-redis

# Xoá bỏ khi ko cần sử dụng lại
docker rm  my-redis

  • Các service khác như mongodb, mysql, postgresql,…. Cũng làm tương tự

  • Giải thích:

    • -p <host-port>:<container-port> mở port trong container để máy host (bên ngoài) có thể truy cập được.
    • -d: chạy ở dạng service ngầm. QUAN TRỌNG không có là ko có chạy được, nó start container lên xong thoát liền.
