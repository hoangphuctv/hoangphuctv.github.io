Go function

Apr 8, 2019 · 2 minute read
goLập trình GO

func là từ khoá dùng để khai báo hàm. Tương tự như từ khoá function trong các ngôn ngữ khác. Đối với bất kì ngôn ngữ nào, chúng ta có một số khái niệm liên quan tới function:

  • Tham số: là input mà chúng ta muốn function xử lí. Ví dụ: print("Hi")
  • Kết quả trả về: là output trả về thống qua tên hàm. Ví dụ: c := sum(a, b)

Cả tham số và kết quả trả về có thể có hoặc không có, có thể có một hoặc có nhiều.

Ví dụ:

Function không có tham số và không có kết quả trả về:

func sayhi() {
    println("Hello world")
}

func main(){
    sayhi()
}

Function có trả kết quả về qua tên hàm:

func sum(a int, b int) int {
    return a + b
}

func main(){
    println("Sum: ", sum(1,2))
}

Nhiều tham số cùng kiểu dữ liệu có thể viết liên tiếp:

func sum(a, b int) int {
    return a + b
}

func main(){
    println("Sum: ", sum(1,2))
}

Có thể return về nhiều giá trị cùng lúc:

func swap(a, b int) (int, int) {
    return b, a
}

func main(){
    a := 1
    b := 2
    b, a := swap(a, b)
    println("a: ", a)
    println("b: ", b)
}

Chỉ định tên biến sẽ return về:

func sum(a, b int) (c int) {
    c = a + b
    // chỗ return này không cần để tên biến.
    return
}

func main(){
    println("Sum: ", sum(1,2))
}
