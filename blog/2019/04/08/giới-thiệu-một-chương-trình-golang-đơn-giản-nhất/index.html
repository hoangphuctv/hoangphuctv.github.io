Giới thiệu một chương trình golang đơn giản nhất

Apr 8, 2019 · 3 minute read
goLập trình GO

Tên chương trình:

In ra màn hình chuỗi “Hello world”

Mục tiêu:

Làm quen với ngôn ngữ lập trình golang:

  • Cấu trúc chương trình.
  • Cách thực thi 1 chương trình.
  • Cách build 1 chương trình.

Code ví dụ

File: ~/go/hello/hello.go

package main
func main() {
    println("Hello world")
}

Chạy chương trình

Tại terminal, cmd hoặc powershell gõ lệnh

$ go run hello.go
Hello world

Chương trình đã xuất ra được dòng chữ Hello world => thành công.

Go là ngôn ngữ biên dịch. Nhưng cách sử dụng lệnh go run như trên, go sẽ thực hiện 2 bước là build và execute. Làm cho chúng ta có cảm giác như đang chạy 1 script php hay python,… kiểu chương trình thông dịch. Nhưng lưu ý: go là ngôn ngữ lập trình biên dịch. Tức là source code sẽ được biên dịch ra file thực thi. Sau này mỗi lần cần chạy chương trình chúng ta chỉ cần thực hiện chạy file thực thi này là được. Xem phần build sau đây để thấy được quá trình này.

Build chương trình

Để build từ source code go sang file thực thi ta sử dụng lệnh build của go:

$ go build hello.go
$ ls 
hello.go    hello

$ chmod +x ./hello
$ ./hello
Hello world

Ta được file hello là file thực thi của chương trình. Nếu là HDH *nix bạn cần chmod quyền thực thi cho file này. Sử dụng lệnh chmod +x ./hello để cấp quyền thực thi cho file hello. Và gõ lệnh ./hello để thực thi file này. Ta cũng sẽ nhận được kết quả giống như gõ lệnh go run.

Đối với HĐH Windows thì file thực thi là file hello.exe

Nếu bạn muốn chỉ rõ tên file của file thực thi. Thì bạn thêm vào tham số -o ten-file

$ go build -o "hello-world" hello.go
$ ls 
hello.go    hello-world

Phân tích chương trình:

Vừa rồi là một chương trình đơn giản nhất dùng để xuất ra câu thông báo Hello world. Các bạn hãy cùng mình lướt qua các vị trí quan trọng nhé:

Từ khoá package

package từ khoá dùng để khai báo một gói của chương trình. Trong go source code phải được đặt trong một package nào đó. go sẽ tìm hàm main trong package main để chạy. Vì vậy package main và func main là bắt buộc phải có cho một chương trìn thực thi. Nếu chỉ là thư viện thì không cần thiết phải có.

  • Cú pháp package tenpackage

  • Vị trí: đặt tại đầu mỗi file.

Từ khoá func

func là từ khoá dùng để khai báo hàm. Tương tự như từ khoá function trong các ngôn ngữ khác. func có thể có tên hoặc không có tên. Xem thêm tại {{ ref “go/go-function.md” }}

Function có tên:

func main(){
    // do something
}

Function không có tên, ta có thể gán vào 1 biến nếu cần:

func (){
    // do something
}

funcHello := func(){
    // do something
}

// call function 
funcHello()

Sử dụng hàm println xuất chuỗi kí tự ra màn hình

Golang có sẵn một số hàm thực hiện việc đơn giản thường dùng. Ví dụ như xuất dòng chữ ra màn hình: print và println.

println("Hello world")

Đây là lệnh sẽ xuất ra màn hình dòng chữ Hello world. Hàm này có thể nhận vào nhiều tham số. Hàm sẽ xuất lần lượt các tham số ra và phân các bởi 1 dấu cách

println("Hello", "world")

Bạn thế hàm này vào hàm ở trên, cũng sẽ nhận được kết quả tương tự.
