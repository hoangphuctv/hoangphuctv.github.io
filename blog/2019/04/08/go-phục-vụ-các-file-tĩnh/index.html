Go phục vụ các file tĩnh

Apr 8, 2019 · 2 minute read
goLập trình GO

Trong quá trình lập trình web, một webserver cần đảm nhiệm các vấn đề sau:

  • Lắng nghe kết nối: server mở port kết nối để các http client (trình duyệt, curl,…) kết nối đến và request các tài nguyên.
  • Xử lí động: xử lí các POST, GET,… request.
  • Phục vụ các file tĩnh (js, css, images,…)

Trong bài này chúng ta sẽ xem package net/http hỗ trợ việc phục vụ các file tĩnh ra sau:

Ví dụ ta có một thư mục static có cấu trúc đơn giản như sau:

static/
├── css
│   └── app.css
├── images
│   └── bg.jpg
└── js
    └── app.js

Ta mong muốn rằng khi user request vào http://localhost:8080/static/css/app.css thì chúng ta sẽ nhận được file app.css

Trước tiên ta tạo ra 1 handler file với tham số đầu vào là 1 đường dẫn.

fs := http.FileServer(http.Dir("static/"))

Kế đó ta thực hiện routing cho các request vào có tiền tố static sẽ được sử lý bởi handler ta vừa khai báo ở trên.

http.Handle("/static/", http.StripPrefix("/static/", fs))

Cuối cùng ta thực hiện lắng nghe kết nối http như ở bài trước.

http.ListenAndServe(":8080", fs)

Code đầy đủ

package main

import (
    "fmt"
    "net/http"
)

func main(){

    fs := http.FileServer(http.Dir("static/"))
    http.Handle("/static/", http.StripPrefix("/static/", fs))

    err := http.ListenAndServe(":8080", fs)
    if err != nil {
        // log error and exit
        panic(err)
    }
}

Lưu nội dung trên vào 1 file ví dụ static-server.go Thực thi bằng lệnh

$ go run static-server.go

Chúng ta mở terminal khác và thử curl vào đường dẫn:

$ curl http://localhost:8080/static/css/app.css

Chúng ta sẽ thấy trả về nội dung của file css. Ở các file khác hệ thống cũng sẽ trả về dữ liệu tương tự.
