Cài đặt PHP

Mar 29, 2019 · 3 minute read
installLập trình PHPphp

Trong 3 HĐH phổ biến là Windows, Linux, MacOS thì trong đó MacOS đã cài đặt sẵn php. còn Linux cũng có thể cài đặt PHP một cách dễ dàng bằng 1 lệnh duy nhất (cài đặt bản stable). Chỉ có Windows là cài đặt hơi mệt hơn. (nghe ngược ngược :D)

Cài đặt PHP trên Windows

Trên HĐH Windows có một số cách để cài đặt PHP:

  1. Cách chính thức nhất và chỉ cài đặt php thì bạn có thể vào thẳng trang web của php tại: https://windows.php.net/download#php-7.2 Bạn tải về 1 file zip trong đó có chứa php. Bạn giải nén nơi đâu tùy ý. Nhưng có thể đặt trong C:\php. Nếu bạn cài nhiều phiên bản php vào có thể đặt trong từng thư mục phiên bản riêng. Ví dụ: C:\php\5.6, C:\php\7.0, C:\php\7.2. Bạn đừng quên làm 1 thao tác quan trọng là add PATH cho thư mục trên nhé. Để bạn có thể sử dụng được lệnh php trên cmd.
  2. Cài đặt các gói xampp, wamp, ampps, easyphp,… các gói này sẽ có đầy đủ PHP, Apache, MySQL, phpmyadmin, Tomcat,… đủ bộ cho bạn cần thiết. Có thể bạn cài luôn một trong những bộ này để sau này học tới phần lập trình website bằng php thì cũng không cần phải cài đặt thêm gì nữa.

Cài đặt PHP trên MacOS

MacOS mặc định đã cài đặt sẵn php, nhưng nếu bạn muốn cài đặt bản mới nhất bạn có thể cài đặt thêm vào hệ thống của mình.
Chúng ta sẽ sử dụng trình quản lí gói brew. Vì vậy nếu chưa cài đặt bạn cần tìm cách hướng dẫn cài đặt brew trước.

Khi đã có sẵn trình quản lí gói brew bạn chỉ cần gõ lệnh sau:

$ brew install php

Lệnh được chạy ở chế độ user thường. brew trên MacOS không cần quyền root để cài đặt.
Sau khi cài đặt, gõ thêm lệnh sau để thay thế php có sẵn trên máy.

$ brew link --force php

Tuỳ vào phiên bản Linux của bạn đang sử dụng mà cách cài đặt cũng khác nhau.
Sau đây nêu một số cách đơn giản của các distro phổ biết cho các bạn có thể dễ dàng nắm bắt.

Linux dòng họ Debian: Ubuntu, Linux Mint,…

Sử dụng trình quản lí gói apt

apt install php

Sau khi thực hiện cài xong. Bạn gõ lệnh php tại terminal

$ php -v
PHP 7.2.12 (cli) (built: Nov  9 2018 11:00:45) ( NTS )
Copyright (c) 1997-2018 The PHP Group
Zend Engine v3.2.0, Copyright (c) 1998-2018 Zend Technologies
    with Zend OPcache v7.2.12, Copyright (c) 1999-2018, by Zend Technologies

Nó ra kết quả gì đó tương tự như trên là được. Nếu nó hiển thị kết quả

$ php -v
-bash: php: command not found

Thì là chưa được, hãy thử tắt và bật lạ terminal. Nếu vẫn không được hãy kiểm tra lại hoặc comment bên đưới để được giúp đỡ thêm.

Để tham khảo thêm đầy đủ về cách cài đặt php và các vấn đề liên quan vui lòng xem thêm tại PHP manual install.
