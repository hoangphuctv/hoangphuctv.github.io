Build extension cho php bất kì trên Linux/MacOS

Mar 31, 2019 · 2 minute read
extensionLập trình PHPphp

Khi bạn cài php từ source mà không phải cài từ trình quản lí gói như apt hoặc yum, brew,… thì khi bạn cần cài thêm 1 extension mới cho php thì ngoài cách bạn thường hay làm (trên windows) là bạn lên mạng; tải file đã build sẵn có định dạng là *.so đối với linux, Mac.

Trước tiên, bạn phải biết đường dẫn tới thư mục bin của php. Ví dụ nó nằm tại /etc/php/5.5/bin/.

Giờ thì bạn mở terminal lên, cd đến thư mục nào đó bạn muốn chứa source (/tmp). Tải về source extension bạn muốn cài đặt (xem tại đây http://pecl.php.net/). Ví dụ như memcache-2.2.6.tgz

$ cd /tmp
$ wget http://pecl.php.net/get/memcache-2.2.6.tgz
$ cd memcache*
$ /etc/php/5.6/bin/phpize
$ ./configure --with-php-config=/etc/php/5.6/bin/php-config
$ make

Bây giờ sau khi lệnh make chạy thanh công, bạn sẽ được file ./modules/memcache.so. Đây là file ext bạn cần. Hãy copy nó tới vị trí file ext bạn mong muốn và thêm chỉ thị load ext trong file php.ini.

Để biết vị trí file php.ini đang load dụng lệnh:

$ php --ini
Configuration File (php.ini) Path: /etc/php/5.5/cli
Loaded Configuration File:         /etc/php/5.5/cli/php.ini
Scan for additional .ini files in: /etc/php/5.5/cli/conf.d
Additional .ini files parsed:

Bạn mở file php.ini lên và thêm vào dòng:

extension=memcache.so

Sau khi thêm xong kiểm tra lại bằng lệnh:

$ php -m ' grep memcache
memcache

Nếu có kết quả hiện ra là extension đã được load thành công.
