Cài đặt flash player cho Ubuntu Linux

Jul 1, 2013 · 1 minute read
Cài đặt - Cấu hình

Mở terminal
sudo apt-get install flashplugin-installer

Cài thêm các thành phần bổ trợ đa phương tiện để play được các video:
sudo apt-get install ubuntu-restricted-extras
